# Awesome MiMo Agent

[English](./README.md) | [简体中文](./README.zh-CN.md)

A curated list of guides for integrating **Xiaomi MiMo** models into popular AI agent and coding-assistant tools.

Each guide walks through installation, configuration, and first run — so you can start using MiMo-V2.5-Pro inside your favorite tool in a few minutes.
