# Awesome MiMo Agent

[English](./README.md) | [简体中文](./README.zh-CN.md)

精选的 **Xiaomi MiMo** 模型接入指南合集，帮助你将 MiMo 接入主流 AI Agent 与编程助手工具。

每份指南都包含安装、配置与首次运行的完整步骤，几分钟即可在你喜爱的工具中用上 MiMo-V2.5-Pro。
